# aerial_warfare
## 简介
这是一款基于pygame的伪雷霆战机的一款2D俯视角闯关射击游戏

目前计划推出更加完善的版本，包括新型初始飞机（发射火箭弹或激光武器）、新型boss、甚至尝试增加局外养成？说不准呢！敬请期待吧！
